# Crawling Multi-Turn Dialogs in Twitter

## Usase

### Crawling

```
python crawl.py /path/to/output/
```

### Cleaning

```
python clean.py /path/to/raw/ /path/to/turn/
```

### Wakatigaki

```
python wakati.py /path/to/turn/ /path/to/wakati/
```
